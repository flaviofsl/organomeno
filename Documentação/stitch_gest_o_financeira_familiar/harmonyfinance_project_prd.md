# Organomeno - Financial Management System

## Project Overview
A ReactJS-based family financial management application designed for clarity, transparency, and collaboration.

## Core Features & Screens
1. **Dashboard**: High-level overview of family finances.
2. **Family Organogram**: Visual mapping of financial relations (Providers vs Dependents).
3. **Transaction Ledger**: Detailed list of all entries (Income/Expenses).
4. **Transaction Detail**: Itemized receipt view for specific purchases.
5. **Account Management**: Linked bank accounts and balances.
6. **Import OFX**: Bank statement reconciliation.
7. **Category Management**: Customizable expense classifications.
8. **Auth Flow**: Login, Register, and Password Recovery.

## Technology Stack
- **Framework**: ReactJS
- **Styling**: Tailwind CSS
- **Design System**: Modern Fiscal Integrity
- **Navigation**: React Router (conceptually)
